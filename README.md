### Install

Add [awesome xonshrc](https://github.com/anki-code/awesome-xonshrc/blob/main/xonshrc) to the end of current xonshrc:
```xonsh
curl -s https://raw.githubusercontent.com/anki-code/awesome-xonshrc/main/xonshrc >> ~/.xonshrc
```
